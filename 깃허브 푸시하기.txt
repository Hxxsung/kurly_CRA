깃환경설정 사용자.이름    git config user.name 'heesung'
깃환경설정 사용자.이메일  git config user.email 'hspark0814@naver.com'

  3. 깃허브 원격지 리포지토리(저장소) Origin(url)

추가: git remote add origin https://github.com/Hxxsung/kurly_CRA.git

* 확인방법 *
git remote -v
